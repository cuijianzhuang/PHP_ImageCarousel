# PHP Image & Video Carousel

一个基于PHP实现的简单照片和视频轮播系统。

## 功能特点

- 支持图片和视频文件的轮播展示
- 简单的后台资源管理系统
- 响应式设计，适配多种设备
- 简单易用的管理界面

## 技术栈

- 前端：HTML5, CSS3, JavaScript
- 后端：PHP
- 数据存储：文件系统

## 安装说明

1. 克隆项目到本地
2. 确保服务器已安装PHP环境
3. 配置web服务器（Apache/Nginx）指向项目目录
4. 访问项目地址即可使用

## 使用说明

### 后台管理

- 访问 文件管理 进入管理界面
- 支持上传、删除、排序媒体文件
- 可设置轮播间隔时间等参数

### 前台展示

- 访问根目录即可查看轮播效果
- 支持自动播放和手动控制
- 适配PC和移动端浏览器
